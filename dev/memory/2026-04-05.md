### 9. 微信二维码展示功能

**问题：** ASCII 二维码在网页中显示错位

**解决方案：** 从控制台输出中提取二维码链接，转换为图片展示

**实现方式：**
1. 添加 qrcode.js 库
2. 监听 SSE 输出，正则匹配链接：`https?:\/\/liteapp\.weixin\.qq\.com\/q\/[^\s\]\n]+`
3. 使用 `QRCode.toCanvas()` 生成二维码图片
4. 页面布局：左侧 2/3 控制台，右侧 1/3 二维码

**关键代码：**
```javascript
// 提取二维码链接
const qrcodeMatch = data.text.match(/https?:\/\/liteapp\.weixin\.qq\.com\/q\/[^\s\]\n]+/);
if (qrcodeMatch) {
    updateWeixinQrcode(qrcodeMatch[0]);
}
```
