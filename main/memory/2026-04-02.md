### Docker 相关知识

1. **docker-compose 转 docker run**
   - 用户有 openclaw-easy-v3 的 compose 配置
   - 转换为独立 docker run 命令

2. **配置警告处理**
   - `feishu-openclaw-plugin: plugin not found` 是无害警告
   - 插件已删除但配置残留，系统自动忽略

3. **Docker 容器动态挂载 U 盘**
   - 运行中容器无法直接添加新挂载
   - 解决方案：预挂载占位目录，用 `mount --bind` 注入
   ```bash
   # 容器启动时预留
   -v /data/usbshare:/media/usb
   
   # U 盘插入后绑定
   mount --bind /media/your-usb /data/usbshare
   ```

## 注意事项

- 容器重启后 Docker 权限可能丢失，需执行：
  ```bash
  docker exec -u root openclaw-easy-v3 sh -c 'echo "docker:x:993:node" >> /etc/group'
  ```
