## 08:54 - 修复天气和早中晚报不发送的问题

### 问题原因
Cron 任务的 `delivery` 配置**缺少 `to` 字段**，导致消息无法投递（"cron announce delivery failed"）。

### 修复操作
为以下 4 个任务添加了发送目标：
- **weather-morning** (7:00) → `ou_7172afb8fa3c2f51bb02b6af097a4b27`
- **news-morning** (8:00) → `ou_7172afb8fa3c2f51bb02b6af097a4b27`
- **news-noon** (12:00) → `ou_7172afb8fa3c2f51bb02b6af097a4b27`
- **news-evening** (20:00) → `ou_7172afb8fa3c2f51bb02b6af097a4b27`

### 修复命令
```bash
openclaw cron edit <job-id> --to "ou_7172afb8fa3c2f51bb02b6af097a4b27" --channel feishu --account notify
```

### 验证
下次定时任务运行时应该能正常发送了。
