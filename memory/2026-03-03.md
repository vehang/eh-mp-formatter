## OpenClaw 升级

### 版本升级
- **升级前**：OpenClaw 2026.2.22-2，Feishu 扩展 2026.2.22
- **升级后**：OpenClaw 2026.3.2，Feishu 扩展 2026.3.2
- **升级方式**：Docker 容器内以 root 用户执行 `npm install -g openclaw@latest @openclaw/feishu@latest`
- **注意事项**：
  - `node` 用户和 `root` 用户 PATH 不同，需要更新符号链接
  - 升级后执行 `ln -sf /home/node/.npm-global/lib/node_modules/openclaw /usr/local/lib/node_modules/openclaw`
  - 容器内没有 `sudo`，必须以 root 身份升级
